# .github — StreamProof org meta

Org-level community health files and cross-repo docs for **Absol-Labs / StreamProof**.

- [`profile/README.md`](./profile/README.md) — the org profile landing page.
- [`ROADMAP.md`](./ROADMAP.md) — cross-repo roadmap, dependency spine, and the
  **prototype cut-line** (how much to build now).
- [`CONTRIBUTING.md`](./CONTRIBUTING.md) — how to develop across the five repos.
- [`docs/codex-playbook.md`](./docs/codex-playbook.md) — the AI-agent (Codex)
  session prompt + playbook for solving issues and raising PRs across repos.
- [`SECURITY.md`](./SECURITY.md) — org security policy.
- [`.github/ISSUE_TEMPLATE/`](./.github/ISSUE_TEMPLATE/) — default issue template for
  org repos that don't define their own.

These files apply org-wide via GitHub's community-health-file defaults.
